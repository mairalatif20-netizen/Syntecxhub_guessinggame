# 🎯 Number Guessing Game

This project is part of the Syntecxhub Internship Program.

## 📌 Description

This is a simple Python-based number guessing game.
The computer randomly selects a number, and the user tries to guess it.

## 🎮 Features

* Random number generation
* User input for guessing
* Hints (Too High / Too Low)
* Attempt counter
* Simple and interactive gameplay

## ▶️ How to Run

1. Open the file `guess_game.py`
2. Run the program in Python
3. Enter your guesses when prompted

## 🧠 How it Works

* The program generates a random number between 1 and 100
* The user keeps guessing until the correct number is found
* The program gives hints after each guess
* Total attempts are displayed at the end
